## Requirements
. NVIDIA GPU

. Linux

## How to install privateGPT

```
cd /path/to/file

sh linux-installer.sh

python3 /home/$USER/privateGPT/privateGPT.py
```

## How to use privateGPT
```
cd /home/$USER/privateGPT
```
### Ingest documents
```
sudo python3 ingest.py
```
### Run privateGPT
```
sudo python3 privateGPT.py
```
